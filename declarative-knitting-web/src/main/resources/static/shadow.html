Shadow Knitting

Produces shadow/illusion knitting patterns from images.

Shadow (or "illusion") knitting is a technique which uses thin stripes and raised ridges to create knitting patterns which are hidden when viewed head on, but appear when seen from an angle. Pat Ashforth and Steve Plummer of Woolly Thoughts are the masters of this technique, and have created some beautiful examples.

This tool takes in picture and uses k-means clustering to divide it into two main colours, then generates a shadow knitting chart based on this image. This will then be downloaded in your browser as a spreadsheet. This will work best with images with obvious foreground and background colours. In the spreadsheet, purl stitches are shown slightly darker than knit stitches - this is to allow the final pattern to be visible in the sheet.

Generating may take a few seconds, please be patient - this is because the generated spreadsheet can be large. Also, please be patient with the error handling - there's quite a few things which can go wrong while trying to get hold of the requested image. The system will make it's best attempt to get the image you request, if it fails please try a different image.

Enter the URL of an image to knit. This should be a direct link, and must end with a file extension (e.g. .jpg)
Enter the width, in stitches, to make the generated pattern. The generator will calculate the appropriate height. Maximum of 500.
